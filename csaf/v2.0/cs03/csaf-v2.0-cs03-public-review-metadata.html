Common Security Advisory Framework Version 2.0

Public Review of Committee Specification 03 as candidate for OASIS Standard

31 August 2022

 

Announcement:

The members of the OASIS Common Security Advisory Framework (CSAF) TC approved submitting Common Security Advisory Framework Version 2.0 Committee Specification 03 to the OASIS membership as a candidate for OASIS Standard on 29 August 2022. See the ballot at https://www.oasis-open.org/committees/ballot.php?id=3725.

The Committee Specification now enters the required 60-day public review that precedes the call for consent.

Public Review Dates:

The public review starts 31 August 2022 at 00:00 UTC and ends 29 October 2022 at 23:59 UTC.

Submit Comments:

Any individual may submit comments to the TC through the use of the OASIS TC Comment Facility, which can be used by following the instructions on the TC's "Send A Comment" page (https://www.oasis-open.org/committees/comments/index.php?wg_abbrev=csaf).

For more information on submitting comments to the TC, please see the Public Comment License section below. Additional information about the specification and the CSAF TC can be found at the TC's public home page: https://www.oasis-open.org/committees/csaf/.
