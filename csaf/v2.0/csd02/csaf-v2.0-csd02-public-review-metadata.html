Common Security Advisory Framework Version 2.0

Public Review of Committee Specification Draft 02

15 April 2022

 

Announcement:

The members of the OASIS Common Security Advisory Framework (CSAF) TC approved submitting Common Security Advisory Framework Version 2.0 Committee Specification Draft 02 for public review on 30 March 2022. See the TC's minutes at https://github.com/oasis-tcs/csaf/blob/master/meeting_minutes/2022-03-30.md#meeting-notes.

The Committee Specification Draft is now available for a 15-day public review.

Public Review Dates:

The public review starts 15 April 2022 at 00:00 UTC and ends 29 April 2022 at 23:59 UTC.

Submit Comments:

Any individual may submit comments to the TC through the use of the OASIS TC Comment Facility, which can be used by following the instructions on the TC's "Send A Comment" page (https://www.oasis-open.org/committees/comments/index.php?wg_abbrev=csaf).

For more information on submitting comments to the TC, please see the Public Comment License section below. Additional information about the specification and the CSAF TC can be found at the TC's public home page: https://www.oasis-open.org/committees/csaf/.
