Common Security Advisory Framework Version 2.0

Public Review of Committee Specification Draft 01

14 August 2021

 

Announcement:

The members of the OASIS Common Security Advisory Framework (CSAF) TC approved submitting Common Security Advisory Framework Version 2.0 Committee Specification Draft 01 for public review on 05 August 2021. See the TC email at https://lists.oasis-open.org/archives/csaf/202108/msg00000.html.

The Committee Specification Draft is now available for a 30-day public review.

Public Review Dates:

The public review starts 14 August 2021 at 00:00 UTC and ends 12 September 2021 at 23:59 UTC.

Submit Comments:

Any individual may submit comments to the TC through the use of the OASIS TC Comment Facility, which can be used by following the instructions on the TC's "Send A Comment" page (https://www.oasis-open.org/committees/comments/index.php?wg_abbrev=csaf).

For more information on submitting comments to the TC, please see the Public Comment License section below. Additional information about the specification and the CSAF TC can be found at the TC's public home page: https://www.oasis-open.org/committees/csaf/.
